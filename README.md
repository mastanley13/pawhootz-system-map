# PawHootz System Map

Static Vite app for the PawHootz system architecture map.

## Local development

```powershell
npm install
npm run dev
```

Vite will print the local URL in the terminal.

## Production build

```powershell
npm run build
```

The production output is written to `dist/`.

## Vercel

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`

No runtime environment variables are required for the current static build.
